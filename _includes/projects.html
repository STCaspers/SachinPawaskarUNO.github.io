Projects

  Mav Basic - The UNO - MavBasic web application attempts to take the pain out of development by providing common basic functionality such as Authentication, User Management, etc. The MavBasic application uses the Laravel web application framework. Documentation for the framework can be found on the Laravel website.
  CoRA - Commingled Remains Analyics - The goal of this project is to build a system that would help make the process of inventorying and cataloguing skeletal elements easier and give forensic anthropologists the capabilities and tools to perform reporting and advanced analytics.
  IoT Analytics - The goal of this project is to build a system that will provide the capability to collect information from IoT devices with sensors directly into the cloud and perform Big Data Analytics on this data. A web application will be built to provide reporting, analytics and visualization of this data, along with administrative capabilites such as management of devices, users, etc.
