Teaching

  ISQA-8950 MIS Capstone - The course consists of a student executed Information Systems design project providing an in-depth practical experience. It typically covers system conceptualization, analysis, and design. It may also involve prototyping. The project will typically not include the actual implementation of the system. This course replaces the MS in MIS comprehensive exam requirement.
  CSCI-3320 Data Structures and Algorithms - This is a core that will cover a number of data structures such as tree, hashing, priority queues and graphs as well as different algorithm design methods by examining common problem-solving techniques.
  CSCI-2240 Introduction to C programming - Programming in 'C' in a UNIX operating system environment; algorithm and program development and file manipulation using 'C'; UNIX-like utility development.
  CSCI-1620 Introduction to C++ programming - Advanced topics in Object oriented programming including data encapsulation, inheritance, polymorphism. Topics in data representation and manipulation, data structures, problem solving and algorithm design.
